# Kawaii Habit Tracker

# Description
This is a monthly habit tracker where the user can track their daily habits.

# Features
- Clickable days
- Highlight days
- Customizable habit name
- days of the week
- Reset button

# Why I made this
I wanted to create something that could help track daily habits

# Tools
- HTML
- CSS
- JavaScript

# Struggles
- One of the main challenges was using the localStorage to manage and store each day

# Learned
- To use localStorage
- Work with date objects

# Disclaimer
- This is not a accurate representation of a calendar.
- The days don't reflect a real-world calendar
